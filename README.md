## Airqualityegg.com

Air Quality Egg dashboard

### Development

#### Add environment variables to .env file

```bash
# Sample .env file
PRODUCT_ID=xxxxxx
API_KEY=xxxxxxx
```

#### Install gem bundler

`gem install bundler`

#### Install rubygems

`bundle install`

#### Start webserver

`bundle exec foreman start`

Visit http://localhost:5000
